# 消息系统
- 支持好友私聊。
- 消息长度限制为 500 字以内。
- 消息窗口尺寸更小，适合悬浮在播放界面上。
- 可拖拽窗口顶部移动位置。
- 可拖拽窗口底部旋转，形成 3D 悬浮效果。
- 收到新消息时，左侧空白处会出现消息气泡。
- 消息气泡可关闭，也可直接回复。
- 社区页提供消息免打扰按钮，开启后不再弹出新消息气泡。
 ###  点赞、听歌时长与认证

- 好友之间可以点赞。
- 客户端会上报听歌时长和当前播放歌曲。
- 服务器根据听歌时长和点赞数自动生成认证：
  - 活跃乐友。
  - 认证乐友。
  - 深度乐友。
- 服务端手动徽章优先级高于自动认证。
- 手动徽章可在服务器后台下发或清除。
### 一起听歌

一起听歌用于让好友在同一首歌上保持播放同步。

### 支持流程：

- 向在线好友发起一起听邀请。
- 对方可同意或拒绝。
- 同意后创建一起听会话。
- 任一成员切换歌曲、播放/暂停或进度变化时，服务器会发布同步事件。
- 其他成员收到同步事件后尝试选中同一首歌并播放。
- 任一成员可退出一起听，服务器会通知会话成员。
- 连麦信令可通过社区服务器中转，为语音或 WebRTC 能力预留通道。
###  如何更新
服务器端可发布新客户端版本：

- 版本号。
- 下载地址。
- 更新说明。
- 文件大小。
- SHA-256。
- 是否允许自动安装。
