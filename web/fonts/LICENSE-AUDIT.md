# FE Monster 字体授权核验

核验日期：2026-07-21

核验标准不是“能否免费下载”或“能否做海报”，而是字体文件能否随 Windows 程序、Web 资源和 Android APK 一起嵌入、复制及再分发。没有明确软件嵌入/再分发许可的字体不会打进发布包。

## 已随程序内置

| 字体选项 | 随包字体 | 状态 | 依据 |
| --- | --- | --- | --- |
| 无界黑 | 标小智无界黑 / LogoSC Unbounded Sans 1.100 | 已内置 | SIL OFL 1.1；官方字体包包含许可副本 |
| Caveat-Regular | Caveat Variable | 已内置 | SIL OFL 1.1；Google Fonts 官方仓库 |
| 得意黑 | Smiley Sans 2.0.1 | 已内置 | SIL OFL 1.1；作者官方仓库明确推荐游戏嵌入 |
| 思源粗宋 | Noto Serif SC Variable（同源兼容字库） | 已内置 | SIL OFL 1.1；与 Source Han Serif SC 共用 CJK 设计源，不冒充原字体文件 |
| 烟波宋 | 猫啃网烟波宋 Regular 1.10 | 已内置 | SIL OFL 1.1；作者发布页明确允许安装到软件/APP并随软件再分发 |

对应许可原文位于 `LICENSES/`。字体文件不得脱离软件单独出售；保留版权信息和许可文本。
